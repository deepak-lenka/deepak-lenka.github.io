# Deepak Lenka Personal Site

This is a dependency-free GitHub Pages site inspired by the clean academic homepage style in the reference screenshot.

## Deploy on GitHub Pages

1. Create a repository named `deepak-lenka.github.io`.
2. Put `index.html`, `styles.css`, and `.nojekyll` at the repository root.
3. Push to GitHub.
4. In the repository settings, enable Pages from the `main` branch root if GitHub does not enable it automatically.

The site will be available at:

```text
https://deepak-lenka.github.io
```

## Edit Content

- Main content lives in `index.html`.
- Visual design lives in `styles.css`.
- The portrait lives at `assets/profile.png` and is displayed at the same 300px square size as the reference page.
- The browser tab icon uses `assets/favicon.png`, generated from the same profile image.
- `PROFILE_README.md` is a GitHub profile README version of the same content.
